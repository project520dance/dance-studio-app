# Product Roadmap

The roadmap describes product direction, not fixed delivery dates. Security, accessibility, testing, and performance remain ongoing priorities.

## Version 0.1 — Completed

- Application shell and public landing page
- Parent, teacher, and admin portal foundations
- Firebase Authentication and Firestore user profiles
- Parent onboarding, family creation, and initial dancers

## Version 0.2 — Completed

- Parent dashboard with real Firestore data
- Family information and dancer summaries
- Centralized dashboard service

## Version 0.3 — Completed

- My Dancers and Add Dancer
- Dancer profiles and structured health information
- Photo placeholder and future team assignment field

## Version 0.4 — Completed

- Admin dashboard
- Firestore aggregation statistics
- Student management
- Admin-role service checks

## Version 0.5 — Planned

- Classes, enrollments, and weekly scheduling
- Teacher assignments

## Version 0.6 — Planned

- Announcements and communication
- Notification preferences

## Version 0.6.1 — Planned

- Schedule Event domain and Firestore model
- Manual Schedule Event administration
- Cancellation and rescheduling foundations
- Parent visibility and future teacher-assignment fields
- Attendance relationship design

## Version 0.7 — Planned

- Family billing, invoices, and tuition
- Stripe payments and payment history

## Version 0.8 — Planned

- Competition Hub
- Teams, schedules, costumes, Nationals, and fundraising

## Version 1.0 — Planned

- Production launch
- Security and accessibility reviews
- Automated tests, monitoring, backups, and recovery
