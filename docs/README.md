# Project 520 Documentation

Project 520 is a modern dance studio management platform for parents, teachers, and administrators.

## Technology

- Next.js App Router
- TypeScript and React
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore
- Stripe planned, but not implemented

## Current Features

- Public landing page and authentication
- Firestore user profiles
- Parent onboarding
- Family and dancer records
- Parent dashboard and My Dancers
- Add Dancer and dancer profiles
- Admin dashboard and student management
- Teacher portal foundation

## Long-Term Vision

Project 520 should become a complete dance studio operating system connecting parents, dancers, teachers, and administrators. It should support schedules, enrollments, attendance, billing, costumes, competitions, waivers, announcements, private lessons, documents, notifications, and a strong Competition Hub.
